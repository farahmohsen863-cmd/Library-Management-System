# Library-Management-System
This project is a simple Library Management System built using python.
## Features
- Add books
- Show all books
- Search for a book
- Delete a book
- Borrow a book
- Return a book
- Show library information

  ## Technologies Used
  - python
  - oop (Onject Oriented Programming)
  - Listis
  - Dictionaries
  - Loops
  - Conditions
